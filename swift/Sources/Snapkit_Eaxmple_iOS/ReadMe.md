## 🚀 구현 내용

### 메인 화면
- ✅ Pull to Refresh 새로고침  
- ✅ **배너 섹션**: API 응답에 따라 노출 여부 결정  
- ✅ **카테고리 섹션**: 4개 고정  
- ✅ **큐레이션 섹션**: 2개 이상 표시 가능  
- ✅ **지역 태그 섹션**: 스크롤 시 sticky 고정  
- ✅ 최상단으로 이동 가능한 Floating 버튼 제공  

### 상세 화면
- ✅ Pull to Refresh 새로고침  
- ✅ 좌/우 스크롤 가능한 이미지 영역  
- ✅ 이미지 우측 하단에 index 표시  
- ✅ 타이틀은 2줄까지만 표시  
- ✅ 기간, 설명, 주소, 운영시간 표시  

### 공통 UI 요소
- ✅ **큐레이션 셀**: 마감일까지 30일 이하일 경우 `D-**` 표시  
- ✅ **태그 리스트**: 메인에서는 1줄만, 리스트 셀에서는 여러 줄 표시  
- ✅ **리스트 셀**: 콘텐츠 영역은 Y축 기준 중앙 정렬, 좋아요 수가 없을 수 있음
